# To Setup 
//Note: ensure Amplify CLI has been installed.<br/>

Run the commands below: <br/>

npm install <br/>

Amplify ini<br/>

Amplify add Auth<br/>

Amplify push<br/>

//Note: ensure aws-exports.js has been generated with your own AWS account details.<br/>

ng serve<br/>
